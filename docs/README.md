# inst377-finalproject

# App Description
People often don’t know what type of clothing to wear depending on the weather, my project
aims to give clear suggestions that match with the user's wardrobe and previously worn clothing
items. Everyone on earth will care about this problem but specifically students, teachers,
business men and women and so on will hopefully be affected by my project. My proposed
system will aim to read the current or proposed days temperature, wind, humidity, and any
chance of precipitation and then suggest various outfits based on the weather information at
hand. 


# Target Broswers
The App's target browsers are mainly IOS and Andriod but It should also work on Mac and Windows.
